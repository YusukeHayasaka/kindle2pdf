Gemini APIキーの取得方法

この拡張機能の文字起こし機能を利用するには、Google AI Studioの無料APIキーが必要です。

手順

  1. Google AI Studio にアクセス
    以下のボタンまたはリンクから公式サイトを開きます。
    https://aistudio.google.com/app/apikey
  2. Googleアカウントでログイン
    普段お使いのGoogleアカウントでログインしてください。
  3. APIキーを作成
    「Create API key」ボタンをクリックします。
    その後、「Create API key in new project」を選択します。
  4. キーをコピー
    生成された文字列（例: AIzaSy...）をコピーします。
  5. 拡張機能に貼り付け
    Kindle to PDF拡張機能の設定画面に戻り、「Gemini API Key」欄に貼り付けてください。
ご注意:
Gemini APIの無料枠には利用制限（1分間に15リクエスト等）があります。
この拡張機能は自動で待機時間を設けて制限内に収まるように設計されていますが、短時間に大量の利用を行うと一時的にエラーになる場合があります。
